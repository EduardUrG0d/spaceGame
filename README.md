# Space Game

Космическая игра в стиле Suika, где вы объединяете планеты и создаете черные дыры!

## Технологии
- TypeScript
- PixiJS
- Vite

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка для продакшена
npm run build
```
