# Oi_iO space merge game

Аркадная головоломка, в которой вы соединяете космические объекты, чтобы создавать всё более крупные тела — от метеорита до чёрной дыры!

## Технологии
- TypeScript
- PixiJS
- Vite

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка для продакшена

```bash
npm run build
```
